# 小红书爆款检索查询模板

本文件提供搜索小红书高热门帖子的查询组合策略。研究阶段开始前先读本文件,然后把 `<地点>`、`<主题>`、`<城市>`、`<对象>` 替换为用户的真实输入,**一次性并行发起多条查询**(不要串行)。

## 核心原则

1. **带"小红书"关键词**:在查询里显式加 `小红书`,优先命中小红书生态内容(含被转载到知乎/Trip.com/马蜂窝的小红书攻略)。
2. **多角度并行**:机位、姿势、穿搭、调色、避雷各角度一次性发出,覆盖更全。
3. **口语化高热词**:小红书用户搜索习惯里这些词命中率高——`攻略`、`抄作业`、`出片`、`机位`、`点位`、`同款`、`保姆级`、`避雷`、`氛围感`、`绝绝子`、`一篇搞定`、`合集`。
4. **snippet 不是依据**:搜到后必须进可访问原文读全文(见主 SKILL.md 第二阶段步骤 2)。

## 按角度的查询模板

### A. 出片机位 / 点位
- `小红书 <地点> <主题> 拍照机位 攻略`
- `<地点> 出片点位 推荐 小红书`
- `<地点> 拍照 打卡点 保姆级攻略`
- `小红书 <地点> 必拍机位 合集`

### B. 姿势 / 动作
- `小红书 <地点> 拍照姿势 抄作业`
- `<主题> 出片姿势 教程 小红书`
- `<对象> 拍照姿势 大全`(如"情侣""闺蜜""单人")
- `小红书 <主题> 摆姿 不尴尬`

### C. 穿搭 / 造型
- `小红书 <地点> 穿搭 拍照 出片`
- `<主题> 穿搭 色系 推荐 小红书`
- `<地点> 拍照 穿什么 显瘦 出片`
- 婚纱类追加:`小红书 <地点> 婚纱照 风格 推荐`

### D. 调色 / 滤镜 / 后期
- `小红书 <地点/主题> 调色 教程`
- `<地点> 拍照 滤镜 VSCO Lightroom 预设`
- `<主题> 胶片感 调色 参数 小红书`

### E. 时段 / 避雷 / 实操
- `小红书 <地点> 拍照 最佳时间 人少`
- `<地点> 拍照 避雷 攻略`
- `<地点> 拍照 预约 门票 注意事项`

### F. 文案(可选)
- `小红书 <地点/主题> 拍照 文案 高赞`

## 缺口补全专用查询

### 缺地点、有主题(基于用户城市)
- `小红书 <城市> <主题> 拍摄地 推荐 合集`
- `<城市> <主题> 出片地点 排行`
→ 从高频出现的地点归纳候选,推荐给用户选。

### 缺主题、有地点
- `小红书 <地点> 拍照 风格 有哪些`
- `<地点> 出片 主题 推荐`
→ 从爆款内容里归纳该地点最火的几种拍法/风格作为主题选项。

## 抓取来源优先级

小红书 App 内页常有登录墙,fetch 多半失败。**优先抓取这些可访问的聚合/转载来源**:

1. 知乎专栏(zhuanlan.zhihu.com)—— 常把小红书攻略二次整理,正文完整
2. Trip.com / 马蜂窝 moments —— 旅游攻略,机位描述详细
3. 各类城市攻略博客 / 旅游站
4. 官方旅游网站(如 meet-in-shanghai.net)

遇到 `xiaohongshu.com/explore`、`/discovery` 链接返回空白或登录墙 → 直接跳过,换下一个来源,**同一链接重试不超过 1 次**。
