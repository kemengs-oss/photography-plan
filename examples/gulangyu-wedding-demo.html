主题与氛围文字定调 + 视觉定调

03 / 11

📖 一句话主题

「南洋老洋楼里的一场私奔」——把鼓浪屿的红砖洋楼、三角梅、海风与夕阳,拍成一组复古电影感、有故事的情侣海岛婚纱。

关键词
复古电影感南洋风情海风松弛金色夕阳
画面想象

傍晚四点后,金黄夕阳打在斑驳红砖墙上,新娘白纱被海风吹起,新郎在旋转楼梯旁伸手相牵——像电影里偷拍到的真实瞬间。

🎨 情绪板 · 配色

主色

象牙白 #f3ece0 · 砖红 #b5613f

辅色 / 点缀

夕阳金 · 海雾蓝 · 三角梅玫红 · 老绿

氛围词

暖调、低饱和、轻微褪色;红砖与白纱的冷暖对比是画面骨架。浪漫、怀旧、松弛,拒绝僵硬棚拍。

服装与造型含道具清单

👗 服装(参考爆款)

  • 主套:白色简约长纱(轻盈易被海风吹起)
  • 便装套:法式碎花裙 / 卡其长裙 + 复古衬衫

💄 妆发方向

  • 底妆:透明感裸妆,海岛潮湿主打哑光防出油
  • 发型:微卷散发 / 低盘发,留发丝给海风吹
  • 夕阳套:适当提高腮红唇色饱和度对抗暖光

🎒 道具与装备清单

  • 复古相机 / 拎咖啡(高频道具)
  • 草帽 / 复古墨镜
  • 平底鞋(沙滩 + 石板路必备)
  • 多套便装(海岛潮湿难干)
  • 防晒伞 + 补妆包 + 定型喷雾
  • 小扇 / 喷雾辅助吹纱

风险与备选出发前要心里有数

06 / 11

⚠️ 来自攻略的高频坑

  • 旺季最美转角人挤人,一处要等半小时才有干净背景
  • 雨季台风季(3–5、7–8 月)海景泡汤
  • 船票需提前实名预约,旺季现场无票;岛上全靠步行
  • 紫外线强、气候潮湿,衣物难干
来源:澎湃 · 挤到鼓浪屿拍婚纱照 · 鼓浪屿婚纱照最全攻略

🛟 备选方案

  • 网红点人多 → 切白色转角 / 冷门小巷 / 内厝澳
  • 阴雨 → 领事馆廊下、洋楼室内窗光拍情绪片
  • 提前 1 个月订工作室与船票,清晨首班抢机位
  • 备好平底鞋、多套便装、防晒伞、补妆包

后期参考调色方向与工具

11 / 11

🎨 调色方向

  • 整体:暖调复古电影感,降饱和、加轻微褪色
  • 红砖墙提亮,夕阳金色温往暖偏移
  • 海景套:提通透度,蓝青往莫兰迪偏
  • 肤色:橙黄通道微调,白皙不蜡黄

🛠️ 推荐工具 / 预设

  • Lightroom 调曲线 + HSL 分通道
  • 胶片预设:Kodak Portra / Gold 暖色系
  • 手机端:VSCO(A 系列)、醒图复古滤镜
  • 统一一套预设套全组,保证调性一致
🎬 交付建议:精修 20–30 张主图统一调色,海景套与洋楼套各保留一组、风格不混搭;输出保留一版原色 + 一版电影感供新人挑选。
